🍽️ Restaurant Reservation & Review Platform – Frontend

This is the frontend of a Restaurant Reservation and Review Platform built using the MERN stack. It allows users to discover restaurants, make reservations, leave reviews, and get personalized recommendations.

🚀 Features
🧾 Reservation Management

Book tables with date, time, and party size

Real-time availability updates

View, update, and cancel reservations

⭐ User Reviews

Add ratings, comments, and photos

Edit or delete reviews

Restaurant owners can respond to reviews

🔍 Search & Filters

Search restaurants by:

Cuisine types

Price range

Location

Advanced filters:

Dietary preferences

Ambiance

Features (outdoor seating, live music, etc.)

🏪 Restaurant Profiles

Detailed restaurant information:

Menu

Photos

Contact details

Operating hours

Restaurant owners can manage listings

⏱️ Real-Time Availability

Prevent overbooking

Live updates based on reservations

🎯 Recommendation System

Personalized restaurant suggestions

Based on user activity and preferences

🛠️ Admin Dashboard

Manage users, restaurants, reservations, and reviews

💳 Payment Integration

Secure payments using Stripe

🧑‍💻 Tech Stack

Frontend Framework: React 19

Routing: React Router DOM v7

Styling: Tailwind CSS (via Vite plugin)

HTTP Client: Axios

Payment: Stripe (@stripe/stripe-js)

Build Tool: Vite
